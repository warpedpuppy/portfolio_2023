# warpedpuppies/portfolio

A personal portfolio website built with React (Create React App).

## Stack

- **Framework**: React (Create React App, JavaScript)
- **Styling**: SCSS (`App.scss`, `src/sass/`)
- **Structure**: `src/pages/`, `src/components/`, `src/site-data/`, `src/layout-templates/`

## Development

```bash
npm install
npm start    # http://localhost:3000
npm run build
```
